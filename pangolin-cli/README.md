# Home Assistant Add-on: Pangolin CLI Client

_Run the Pangolin CLI inside Home Assistant to securely connect to remote private resources._

This add-on allows you to run the Pangolin CLI as a client within Home Assistant, making it easy to access and manage your private services directly from your Home Assistant instance.

## Features

- Runs the Pangolin CLI in a lightweight container
- Connect to private resources securely
- Fully compatible with Home Assistant add-on environment
- Easy to configure and start using
